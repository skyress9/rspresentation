React

History of React

  • React is an open-source Javascript library used to create components and user interfaces.
  • React is commonly used as a codebase for SPAs (single-page applications).
  • React was finally introduced in May 2013.
  • In 2014, React really started to gain momentum as important new features were added, such as React Developer Tools as an extension.

Functional and class components

  • Functional component are literally simple JavaScript function. A component displays user interface code, often a JSX expression or raw HTML code.
  • Class component, on the other hand, are written based on regular ES6 JavaScript classes.
  • Many developers and libraries are shifting to using functional components.

Advantages of React

  • Easy to learn.
  • High level of flexibility and maximum responsiveness.
  • Virtual DOM.
  • Combined with ES6/7.
  • Linking data from larger to smaller.
  • A 100% open-source JavaScript library.
  • Incredibly lightweight.
  • Migration between versions is usually very easy.
